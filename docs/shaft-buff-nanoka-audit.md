# Shaft Buff Nanoka 核对报告

核对日期：2026-07-07

数据来源：

- `https://nte.nanoka.cc/`
- `https://static.nanoka.cc/manifest.json`，NTE 当前 `latest/live` 均为 `1.2`
- `https://static.nanoka.cc/nte/1.2/zh/weapon/{id}.json`
- `https://static.nanoka.cc/nte/1.2/zh/console.json`
- `https://static.nanoka.cc/nte/1.2/zh/character/{id}.json`

口径：

- 弧盘按要求只要精 1 或精 5 有一档能对上我方 md / 配置，即可记为可验收。
- 本报告把 `buffs.json` 的 `duration.ticks` 按 0.1 秒换算成秒后和站点文本比较。
- `Q` 对应站点「极轨终结」，`E` 对应「变轨技能」，`援护` 对应「援护技」。
- `通过` 表示触发、持续、增益、层数主要字段可对上；`部分通过` 表示核心数值能对上但条件、冷却、叠层方式或作用对象不完整；`需改` 表示关键持续、数值、层数或触发条件不符。

## 建议改动优先级

1. 先改确定错误且影响大的条目：`危险游戏`、`小小大冒险`、`最后一朵玫瑰`、`无畏之棉`、`时间大盗`、`现实避难所`、`凶猛之棉`、`极速之棉`。
