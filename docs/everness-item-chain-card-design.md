# 异象对决设计文档

日期：2026-06-15

> 本文件是异象对决当前唯一现行设计文档。规则、术语、卡牌效果、异能者职责、构筑口径和实现取舍均以本文为准；测试流程和迭代报告只保留在 `docs/duel-balance-iteration.md` 及其报告目录中。

## 设计目标

本批设计用异象道具承担启动、调度、送墓、回收、小干扰和桥接功能；异能者作为异能者编队，在复杂素材条件满足后把这些资源转化为环合、铺场、清场或强力中转。

核心体验对标游戏王的定向展开链与影之诗的体系节奏：低费启动，中期围绕素材被解、被保护、被回收展开博弈，5 到 6 费让异能者和高费道具把优势一次性放大。

## 基础对局规则

- 当前目标版本使用单战场牌桌，只保留一个主战场，同时保留“战场特性”制造每局差异。
- 常规对局为 6 回合；每回合可用能量等于当前回合数，最高 6 点。
- 主牌组为 10 到 20 张异象道具，同名道具最多 3 张。推荐预组使用 20 张，轻量构筑使用 12 到 16 张，极简测试至少 10 张。
- 异能者作为异能者编队，按单张配置，不使用同名重复。
- 异象道具基础战力必须大于 0。道具被减为非正战力时破碎，移动至墓地。
- 异能者拥有基础战力。异能者战力非正时返回异能者编队。
- 异能者在已经登场过的回合，仍可以继续消耗素材结算效果并吞噬战力；该效果统一称为“共鸣”或“再共鸣”。
- 异能者登场条件通常需要 2 个素材；轻量支援异能者可以只消耗 1 个素材，但其效果必须限 1 次且基础战力为 3；王牌可以需要 3 个素材或附加条件。
- 成功登场异能者必须带来明显收益，如铺场、清场、强力中转、资源回收或终端爆发。

## 回合阶段

阶段转换不需要额外报幕，前端保持当前的自然过渡表现；教学文档需要明确阶段名称和阶段职责。

| 阶段     | 职责                                              |
| ------ | ----------------------------------------------- |
| 开始阶段   | 持续伤害等倒计时类标记层数 -1；比较双方当前总战力并确定本回合结算先后手。          |
| 抽卡阶段   | 双方玩家各执行 1 次通常抽卡。                                |
| 部署阶段   | 玩家自由选择要部署的异象道具和异能者共鸣；确认部署完毕前可以自由撤回。             |
| 素材消耗阶段 | 双方玩家确认部署完毕后进入素材消耗阶段，异能者共鸣效果消耗的素材在此时消耗，素材移动至墓地。  |
| 揭示阶段   | 素材消耗结束后，由结算先手开始将全部部署卡牌揭示，直到该方没有未揭示的卡牌；再由结算后手揭示。 |
| 结束阶段   | 持续伤害标记等非环合自动效果在此时生效；环合标记本身不自行结算。                |

## 核心流派定义

| 流派 | 属性      | 核心设计理念                                                                               |
| -- | ------- | ------------------------------------------------------------------------------------ |
| 创生 | 光 + 灵   | 道具特色是以「都市活力」和「速食早餐袋」作为双启动点，围绕耗材回手、墓地回收和减费复用养大关键单位。       |
| 延滞 | 光 + 相   | 拖延对手行动，以影响对手的效果为主，并能真正阻止或拖慢异能者登场；新增道具特色是以个人材料直接影响战场和对手行动，封素材、换控制、加税和手牌干扰，但基本缺乏回收能力。 |
| 浊燃 | 咒 + 暗   | 节奏卡组，持续建立优势并压制对手，在中期滚出节奏差；新增道具特色是材料轴与番茄轴并行，自选箱启动材料送墓收益，西红柿启动番茄链。                     |
| 黯星 | 暗 + 魂   | 爆发式卡组，前期乏力，依靠设置黯星、保护延迟爆点和后期引爆一波返场；新增道具特色是围绕「本性像素」蓄能、生成和返场展开。                         |
| 盈蓄 | 创生 + 延滞 | 创生与延滞的融合打法，不是独立道具体系；核心是费用魔术，每回合部署并揭示大量卡牌，同时靠延滞压缩对手部署窗口。                                 |
| 失谐 | 浊燃 + 黯星 | 浊燃与黯星的融合打法，不是独立道具体系；核心是纯粹控制，期望对手空场，再依靠后期高费异象道具取胜。                                    |

## 环合设置权限

- 基础环合只能由对应属性相关异能者设置：灵设置创生，相设置延滞，咒设置浊燃，魂设置黯星。
- 浔是当前唯一明确的光属性创生终端例外。
- 光、暗和泛用异能者可以调度、保护、回收、修正素材或促成融合，但不能无条件跨属性设置基础环合。
- 鉴定师是泛用角色，可以作为创生队人选，也可以进入延滞队；其环合来自消耗素材映射，消耗光属性 + 灵属性素材时设置创生，消耗光属性 + 相属性素材时设置延滞，不能脱离素材任意选择环合。
- 达芙蒂尔主要属于失谐队；进入黯星队只是因为当前黯星可用魂属性角色不足，作为暗属性借位角色负责桥接和失谐结算，不作为黯星主设置者。

## 异能者效果描述口径

- 异能者卡面不写实现字段名或程序内部概念，例如“素材标记”“火种/失序素材”“暗尘/信号/器物素材”。需要统计对象时，必须写成玩家能在牌面直接看到的条件，例如“己方战场上表侧的光属性或灵属性道具”。
- 不单独写“保护”。需要保护时，必须写明“给予护盾，持续一回合”；护盾的含义以“时机口径”中的护盾定义为准。
- 不写“低战力”。需要自动指定弱单位时，写“当前战力最低的表侧单位”；需要指定强单位时，写“当前战力最高的表侧单位”。若存在并列，由该效果操作者选择 1 张。
- 不写“盈蓄数”。需要统计盈蓄牌时，写“本回合己方已经揭示过的「盈蓄」卡牌数量”或“手牌中的「盈蓄」卡牌数量”；需要统计盈蓄标记时，写“己方盈蓄标记层数”。盈蓄牌和盈蓄标记是两种对象，不能混写。
- 灵属性异能者只直接作用于光属性和灵属性相关道具、创生、护盾与回收，不在效果中引用相、咒、暗、魂属性。
- 相属性异能者只直接作用于光属性和相属性相关道具、延滞、战场控制、费用增加与素材封锁，不在效果中引用灵、咒、暗、魂属性。
- 咒属性异能者只直接作用于咒属性和暗属性相关道具、浊燃、送墓、削战力与墓地收益，不在效果中引用光、灵、相、魂属性；安魂曲可以通过其专属桥牌把浊燃与失谐结算连接起来。
- 魂属性异能者只直接作用于暗属性和魂属性相关道具、黯星、引爆、「本性像素」资源与谕石链，不在效果中引用光、灵、相、咒属性。
- 光属性桥接异能者除浔外，必须平等看待灵属性与相属性：卡面要同时写清“光 + 灵”和“光 + 相”两条路线，不偏向其中一边。
- 暗属性桥接异能者除安魂曲专属桥外，必须平等看待咒属性与魂属性：卡面要同时写清“暗 + 咒”和“暗 + 魂”两条路线，不偏向其中一边。

## 时机口径

- 共享回合：双方在同一个回合内同时部署、同时消耗素材，再按结算先后手揭示。效果持续时间优先使用“本回合”和“持续一回合”，不写“己方回合”或“对手回合”。
- 回合开始：每个回合开始时，先比较双方当前总战力并锁定本回合结算先手，然后进入部署阶段。本回合结算先手一旦锁定，不会因之后部署、素材消耗、战力变化或揭示效果而改变。
- 结算先手：回合开始时当前总战力更高的一方成为结算先手；若战力相同，随机决定结算先手。随机结果写入日志。
- 当前总战力：统计双方场上稳定单位的当前战力；不统计墓地、手牌、牌库、宣言牌、待揭示道具和已被素材消耗的道具。战斗版图不保留顶部栏，双方当前总战力、实时领先方、本回合结算先手、手牌数和牌库厚度应分别集中显示在左/右侧栏。
- 部署锁定：玩家从手牌打出道具时只支付费用并把卡牌置入部署中，不立刻结算任何卡牌效果。双方完成部署后锁定本回合部署记录，之后才进入素材和效果结算。
- 揭示：异象道具统一使用“揭示”作为卡面结算时机，不再设计单独的“使用”词条。结算先手方先按部署顺序揭示并结算自己的全部本回合部署道具；随后结算后手方按部署顺序揭示并结算自己的全部本回合部署道具。
- 宣言：若道具效果需要检视牌库、选择目标或二选一，操作者必须在部署该牌时完成宣言并锁定选择；揭示阶段只实现已宣言的选择，不再重新选择。候选列表只对操作者可见，最终影响的牌名公开并写入日志。
- 宣言失效：若揭示时宣言对象已离开指定区域、失去合法属性或不存在，则该部分不结算，不允许改选。
- 检视写法：卡面需要把检视与选择写明，例如“宣言：检视牌库，选择 1 张饮料或食物。揭示：将宣言牌加入手牌。”固定牌名不需要检视时，可以直接写“揭示：从牌库将 1 张「环石」加入手牌。”
- 复制规则：复制含有宣言目标的效果时，复制件不能弹出新的选择。卡面必须写明自动目标规则；若没有合法自动目标，则不复制该指向部分。
- 卡名：指名具体道具或异能者时统一使用「」；“将「甲」或「乙」加入手牌”表示从当前合法对象中二选一，不是随机选择。
- 素材：只有进入本回合前已经稳定在场的异象道具可以作为异能者素材。本回合部署中的道具、由本回合效果部署或生成的道具，都要等本回合完全结算后才稳定入场，不能被选为当回合素材。异象道具战力被减为非正时移动至墓地，素材条件会因此被打断。
- 结算顺序：开始阶段处理持续伤害等倒计时标记并确定结算先后手 -> 抽卡阶段双方通常抽卡 -> 部署阶段双方部署道具并声明共鸣 -> 素材消耗阶段双方消耗共鸣素材 -> 揭示阶段结算先手方揭示全部道具 -> 结算后手方揭示全部道具 -> 结束阶段结算持续伤害标记等非环合自动效果。
- 揭示影响：揭示效果可以影响和检测本回合已经揭示并结算过的卡牌和事件；不能影响尚未揭示的待揭示道具，也不能回溯改变已经完成的素材消耗、宣言选择或已经结算完毕的效果。由揭示效果部署到战场的道具加入本方本回合揭示队列尾部，稍后同样揭示并触发揭示效果；由效果生成的非道具衍生物或环合标记不进入揭示队列。
- 表侧表示：所有影响战场卡牌的效果只能选择或影响表侧表示的卡牌，不能影响背面、部署中或尚未揭示的卡牌。因此，结算先手方影响对手战场卡牌时，只能影响对手进入本回合前已经表侧在场的卡牌；结算后手方影响对手战场卡牌时，可以影响对手本回合已经揭示并表侧在场的卡牌。
- 本回合记录：卡牌效果写“本回合”时，可以读取部署锁定记录、素材消耗记录、已经存在的持续标记，以及按结算先后手已经发生的本回合揭示结果；不能读取尚未揭示的卡牌效果。
- 揭示效果：不再设计等待未来事件结算的卡面机制。道具揭示时应立刻结算，或只影响本回合已经部署、揭示、加入手牌、进入墓地、被消耗或仍在战场上的卡牌。
- 护盾：护盾是一种短持续保护状态，只能给予当前战场上的单位。拥有护盾的单位持续一回合不会受到伤害；伤害指会降低战力或使其因战力变为非正而离场的效果。护盾不阻止回手、费用变化、素材消耗、附着、复制或检索。
- 抽牌表现：回合开始补牌改为双方各从牌库抽 1 张，不再进行牌库顶二选一。战斗版图必须显示双方牌库，牌库区需要根据牌库厚度有视觉差异；回合开始双方补牌动画同时播放，揭示阶段从牌库加入手牌也要播放牌库到手牌的抽牌动画。
- 揭示动画：揭示阶段中卡牌效果影响其他战场卡牌时，必须播放从效果来源到受影响卡牌的箭头动画。结算先手方的全部揭示完成后，才轮到结算后手方揭示。
- 盈蓄：盈蓄的 -1 费用返还是费用规则，不视为卡牌效果；它可以在部署阶段即时改变可用能量，但不会引发检索、部署、战力变化或其他效果。
- 环合：异象道具原则上不直接设置基础环合。创生、延滞、浊燃、黯星主要由异能者共鸣设置。卡面只写“设置环合：某环合 N 层”，不写“在对手侧设置”；设置的环合记录在操作者己方战场上。环合标记只记录层数，本身不在回合开始减少，也不在结束阶段或决胜回合自动产生加战力、减战力、加税或爆发效果；只有具体卡牌或异能者效果读取层数时才结算收益。
- 融合结果：盈蓄和失谐是融合标记，不是普通基础环合。创生层数增加时，若己方同战场已有延滞标记，增加等量盈蓄标记；盈蓄标记不超过创生与延滞两者层数的较小值。浊燃或黯星层数增加时，若己方同战场已有另一方标记，增加等量失谐标记；失谐标记不超过浊燃与黯星两者层数的较小值。盈蓄牌仍是 -1 费衍生牌，由具体卡牌或异能者效果生成、部署或揭示，不等同于盈蓄标记。
- 持续伤害标记：诛恶护持、噩梦、判予秋等持续伤害标记显示在环合标记后方，只显示标记名称和层数，不写“持续伤害”字样。它们不是环合；回合开始层数 -1，结束阶段按各自规则生效。
- 字段：所有检索、送墓和回收都按真实字段执行。宽泛检索只使用类别、属性、费用和明确道具名；不再写“关联某某角色的道具”。类别只有一层，可以来自官方类别，也可以把礼物、饮料、食材这类清晰玩法分类升格为类别。卡牌可见描述只写中文术语，不展示实现字段名。

## 已核验世界观链路

本节来自本地异环道具工具查询结果，可作为卡牌效果的链路依据。

| 链路                                              | 设计用途                                      |
| ----------------------------------------------- | ----------------------------------------- |
| 畅爽焕能 -> 都市活力                                    | 创生双启动链，都市活力定向找饮料或耗材，畅爽焕能复用都市活力并带动低费耗材。 |
| 来自「伊波恩」的蛋糕 -> 墓地中的饮料或耗材道具             | 创生、浊燃、盈蓄、失谐共享的中转桥。                        |
| 吃薯片专用洗指机 -> 阿德勒、达芙蒂尔、娜娜莉、塔吉多                    | 回手保护与跨属性桥，适合防干扰和融合套。                      |
| 西红柿 -> 番茄百分百 -> 番茄全家桶 -> 安魂曲                    | 浊燃番茄启动轴，从食材、食物过渡到礼物，服务安魂曲与失谐。             |
| 初级角色异能材料自选箱 -> 初次的期许、思维的同频、水波的迟疑、新兵的怯懦、雏鸟的希冀      | 延滞个人材料调度链，偏向直接影响战场和对手行动；该自选箱带有「不可构筑」标签。 |
| 初级异象材料自选箱 -> 失落絮语、褪色掠影、模糊数符、悬想幻妄                | 浊燃材料启动轴，一送墓一入手，材料拥有额外墓地收益。                |
| 异象追猎材料自选箱 -> 倾诉花种、妄想彼端的一页、水月色拨片、遗失的钱包、斑斓的票根、海之泪 | 中后期定向终端素材。                                |
| 环石 -> 本性像素；去噪底液 -> 本性像素；织梦结 -> 本性像素             | 黯星像素蓄能链，所有资源转换优先围绕本性像素展开。                 |
| 聆谕水晶 -> 谕石                                      | 黯星检索与爆发定位链。                               |
| 创生耗材复用链 -> 速食早餐袋、吃薯片专用洗指机、通勤公文包、遗失的钱包           | 创生专属的耗材回手、墓地回收和减费复用体系；延滞不再依赖这条链。          |

## 通用检索维度

道具定义只保留少量清晰维度，避免额外设计一层人工分类：

| 字段    | 示例                                  | 用途         |
| ----- | ----------------------------------- | ---------- |
| 属性字段  | 光、灵、相、咒、暗、魂                         | 异能者素材条件。   |
| 类别字段  | 食物、食材、饮料、礼物、耗材、材料、货币、家具、任务、钓鱼、重要、其他 | 定向检索与类型协同。 |

## 四系道具边界

四个基础体系的异象道具必须有明确分工，避免所有牌组都靠同一套检索、回手和削战力模板运转。

| 体系 | 道具边界                                                                                 |
| -- | ------------------------------------------------------------------------------------ |
| 创生 | 启动点固定围绕「都市活力」和「畅爽焕能」；优势来自耗材反复回手、从墓地返回手牌、下次部署费用 -1 和多次利用素材。创生可以保护己方素材，但不应主动承担延滞式控场。   |
| 浊燃 | 必须保留材料轴和番茄轴。材料轴由「初级异象材料自选箱」启动，至少一部分材料拥有“从牌库置入墓地时”的额外收益；番茄轴由「西红柿」启动，负责暗属性素材、安魂曲桥和失谐桥。 |
| 延滞 | 道具以光/相材料为主，每张材料应有更强的个人特色，可以直接交换控制权、降低战力、提高素材消耗门槛、干扰手牌或影响对手部署。延滞不提供稳定回手、墓地回收或耗材减费循环。  |

## 盈蓄：创生 + 延滞融合

盈蓄不是独立道具体系，不再存在“盈蓄件”或“盈蓄税”。它由创生链的低费铺场和延滞链的控制自然叠出：创生层数增加时，若己方同战场已有延滞标记，增加等量盈蓄标记，且不超过创生与延滞两者层数的较小值。盈蓄牌仍是 -1 费 0 战力衍生牌，只有具体卡牌或异能者效果写明生成时才加入手牌。盈蓄的 -1 费用返还是费用规则，不视为卡牌效果；它可以在部署阶段即时改变可用能量，但不会引发检索、部署、战力变化或其他效果。

## 失谐：浊燃 + 黯星融合

失谐也不是独立道具体系，而是浊燃与黯星交汇产生的融合标记。浊燃或黯星层数增加时，若己方同战场已有另一方标记，增加等量失谐标记，且不超过浊燃与黯星两者层数的较小值。失谐标记本身不自行结算，具体收益由达芙蒂尔、安魂曲或终端道具读取。道具设计重点是让安魂曲、达芙蒂尔、阿德勒、法帝娅能把浊燃素材和黯星资源接在一起。

## 构筑建议

主牌组允许 10 到 20 张异象道具，同名道具最多 3 张。异能者编队仍按异能者单张配置，不使用同名重复。

这个规则更适合当前目标：玩家可以带 3 张核心启动牌提高爽感和稳定性，也可以少带牌降低学习成本；代价是牌库更快耗尽、转换轴更少、后期资源更薄。推荐预组仍使用 20 张，用来展示多条链路和吃干扰后的转线；玩家自筑和早期测试允许 10 到 16 张。

上面的 8 张表是核心样张，不是完整牌池；完整 20 张预组按“5 个功能位 × 每位 4 张牌”扩展。这样玩家只需要理解 5 类作用，不需要背 20 个完全不同的阅读题；但每个功能位内部目标、费用、时机不同，对局不会固化。

### 重复与少带规则

| 规则    | 建议                                                        |
| ----- | --------------------------------------------------------- |
| 主牌组数量 | 10 到 20 张异象道具。推荐预组 20 张，轻量构筑 12 到 16 张，极简测试 10 张。         |
| 同名上限  | 每种异象道具最多 3 张。核心启动可带满 3 张，高费终端通常 1 到 2 张。                  |
| 同名效果  | 不限制同名道具一回合揭示次数；同名最多 3 张本身就是稳定性上限。                         |
| 少带收益  | 起手和调度更稳定，更容易打出主轴。                                         |
| 少带代价  | 转换小轴更少，被干扰后更难妥协转线；牌库耗尽后，检索和从牌库部署会失效，只能依赖本体系已有的回收、返场或手牌存量。 |
| 构筑底线  | 10 张牌至少需要 4 个不同名称，避免 3 张核心牌堆满后过度单调。                       |

推荐预组的 20 张牌中建议有 12 张效果读法稳定的“低记忆牌”，5 张转换牌，3 张高爆点牌。10 到 16 张轻量构筑则优先保留 6 到 9 张低记忆牌、2 到 4 张转换牌、1 到 3 张高爆点牌。低记忆牌可以共享句式，但不能完全重复效果。例如同为检索牌，「都市活力」找饮料，「初次的期许」找「思维的同频」或「水波的迟疑」，「初级异象材料自选箱」按固定名单一送一拿，它们的读法一致但目标和后续路线不同。
