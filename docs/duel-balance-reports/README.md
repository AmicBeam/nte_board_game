# 对局平衡迭代报告目录

本目录保存每轮迭代报告。现行平衡验收以创生、延滞、浊燃三角看板数据为准；浏览器验收用于确认真实牌桌 UI、动画、交互和日志表达。

命名规则：

- `iteration-01.md`
- `iteration-02.md`
- `iteration-03.md`
- `iteration-04-browser-final.md`

每份报告必须包含：

- 本轮规则和卡组改动。
- 标准三角看板命令、seed、samples 和输出路径。
- 创生、延滞、浊燃的胜率、首个异能者回合、平均登场数、全部登场率。
- 看板验收状态、失败项和关键明细。
- 对局、异能者、卡牌、拖延页暴露的问题归因。
- 下一轮改动清单。
- 是否满足 `docs/duel-balance-iteration.md` 中的看板验收指标。
