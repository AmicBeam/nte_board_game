# Iteration B05：基础四套新规则验证

日期：2026-06-06

## 本轮目标

验证新规则与新牌桌表现下的 4 个基础套路：创生、延滞、浊燃、黯星。

重点检查：

- 回合开始根据当前总战力锁定结算先手，并在侧栏显示。
- 回合开始补牌改为牌库抽牌动画，不再出现牌库顶二选一。
- 揭示阶段从牌库加入手牌时播放抽牌动画。
- 卡牌效果影响其他战场牌时播放箭头动画。
- 对手手牌以卡背朝上显示。
- 双方牌库可见，并根据牌库厚度显示不同视觉状态。

## 规则与 UI 实现记录

- 后端在回合开始锁定 `settlement.first_side`，揭示阶段按结算先手方整批揭示后再轮到后手方。
- 回合补牌改为双方自动从牌库抽 1 张；旧 `draw` 选择会被迁移为自动抽牌并不再公开给前端。
- 前端移除顶部栏，左侧栏显示我方能量、战力、结算先手、回合和牌库；右侧栏显示对手信息、牌库、暗置手牌。
- `draw_card`、`impact_arrow`、`consume_material` 都能在浏览器中被观察到动画。
- 清理旧术语：运行文案不再出现“牌库顶二选一”“沉入牌库底”“埋下黯星”。

## 本轮调整

延滞对浊燃时容易被早期浊燃连续削素材，导致哈索尔之后仍难以阻止浊燃滚雪球。

已调整：

- `delay_control` 从“设置延滞并使最高战力对手 -1”提升为“-2”。
- 影响卡：通勤公文包、排队丝线、锈钟、蓝站台、最终通知。

观察：

- B2 的影响箭头从 3 次提升到 9 次，延滞对素材的实际干扰感增强。
- B1 变成创生小劣，说明延滞的控制感已经足够明显；后续若继续加强延滞，应避免直接加伤害，优先改成“对浊燃专用的节奏阻断或回手/护素材”。

## 结论

基础四套在新规则和新牌桌下可玩通过：中前期有启动、检索、打断素材和结算先手博弈；3 到 4 费可稳定出现异能者素材消耗；4 到 5 费开始出现明显滚雪球和爆发。

仍需后续观察：

- 创生是入门套，评分偏低但体验符合“低干扰、靠身材与保护取胜”。
- 延滞对浊燃的结构仍偏吃力，后续更适合加入抗浊燃的小轴，而不是继续提升通用压战力数值。
- 黯星与创生的后期爆点都偏大，适合保留作为爽点，但后续六套全量平衡时需要关注它们对进阶套的压制力。
