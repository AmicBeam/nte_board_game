# B08：环合层数底层调整脚本轮

日期：2026-06-15

## 本轮目标

按最新设计收口环合底层：

- 创生、延滞、浊燃、黯星标记本身只记录层数，不再在回合开始减少，也不再自动结算加战力、减战力、加税或决胜爆发。
- 创生层数增加且同战场已有延滞时，生成等量盈蓄标记，上限为创生与延滞层数较小值。
- 浊燃或黯星层数增加且同战场已有另一方时，生成等量失谐标记，上限为浊燃与黯星层数较小值。
- 异能者制造基础环合的层数普遍从 2 层降为 1 层。
- 本轮暂不改王牌按环合层数增强的效果，只保留制造层数变化。

## 验证范围

## 结论

- 只降低制造层数且移除环合自带效果后，总体评分结构与 B07 很接近，没有出现单套牌崩盘。
- 创生仍能在 3 费稳定进入首个异能者窗口；失谐面对创生时仍被拖到 5 费首登场。
- 延滞最低分没有改善：它能拖慢自己或对手的登场窗口，但建立优势仍慢于浊燃和创生的资源推进。
- 浊燃、黯星仍偏高分。基础环合不再自动造成伤害后，它们主要依赖道具削战力、持续伤害标记和角色读取层数维持强度。
- 盈蓄/失谐融合标记已通过单测覆盖，但脚本日志尚未专门统计标记生成次数；后续评测脚本应加入标记增长指标。

## 本轮改动

- 基础环合不再回合开始衰减，也不在结束阶段或决胜回合自动结算。
- 延滞标记不再作为通用部署加税来源。
- 盈蓄、失谐改为底层融合标记，由基础环合层数增长时按上限自动生成。
- 娜娜莉、浔、哈索尔、鉴定师、阿德勒、白藏、法帝娅等制造基础环合的效果调整为普遍 1 层。
- 白藏与浔未采用“按环合层数增强王牌效果”的新改动，保留原效果框架。
- 设计真源 `docs/everness-item-chain-card-design.md` 已同步新规则。

## 下一轮重点

- 优先处理延滞：增强前置素材干扰的可见反馈，而不是简单抬终局点数。
- 评测脚本补充盈蓄/失谐标记生成次数、首登场回合差和总共鸣次数差。
- 浏览器可访问性恢复后，重新做正式 UI 验收，重点看融合标记展示、持续伤害标记衰减和素材失败原因。
